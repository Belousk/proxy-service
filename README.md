# Proxy Service ISMPO

Система управления прокси-серверами с личным кабинетом, авторизацией и десктопным клиентом. Проект построен на современной архитектуре: FastAPI (Backend), Vue 3 + Vuetify (Frontend) и PostgreSQL.

---

## 🚀 Быстрый запуск (Docker)

Весь проект (Бэкенд, Фронтенд, База данных) настроен на запуск через Docker Compose одной командой.

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. В корне проекта выполните:
   ```bash
   docker-compose up --build
   ```

После запуска сервисы будут доступны по адресам:
- **Frontend**: [http://localhost:3000](http://localhost:3000)
- **Backend API**: [http://localhost:8000](http://localhost:8000)
- **Документация Swagger**: [http://localhost:8000/docs](http://localhost:8000/docs)

## 🛠 Инструкция по использованию

### 1. Регистрация
1. Перейдите на страницу регистрации: \`http://localhost:3000/register\`.
2. Введите ваш **Email**, **Пароль** и **Подтверждение пароля**.
3. После создания аккаунта вы увидите сообщение об успешной регистрации. Система автоматически сгенерирует ваш первый ключ активации.

### 2. Получение и обновление ключа
1. Войдите в систему через страницу \`/login\`.
2. Перейдите в **Личный кабинет** (кнопка "Профиль" или адрес \`http://localhost:3000/profile\`).
3. В кабинете отображается ваш текущий **Ключ активации (UUID)**.
4. Если вам нужно аннулировать старый ключ, нажмите **"Обновить ключ"**. Новый ключ будет сгенерирован мгновенно и отправлен на вашу почту (через Celery).

### 3. Добавление прокси (Администрирование)
1. На главной странице (\`http://localhost:3000/\`) нажмите кнопку **"Добавить сервер"**.
2. Укажите название, IP-адрес, порт и выберите протокол (SOCKS5/HTTP).
3. После создания прокси появится в списке и станет доступным для десктопного клиента.

## 💻 Запуск десктопного приложения

Десктопный клиент написан на **Python + PyQt5**. Он позволяет подключаться к вашим прокси-серверам.

1. Убедитесь, что у вас установлен Python 3.10+.
2. Откройте новый терминал и перейдите в папку с клиентом:
   ```bash
   cd desktop_client
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```
5. В интерфейсе приложения введите ваш **Ключ активации** из личного кабинета для синхронизации серверов.

## 🏗 Технологический стек
- **Backend**: FastAPI, SQLAlchemy (Async), PostgreSQL.
- **Frontend**: Vue 3, Vite, Vuetify 3, Axios.
- **Background Tasks**: Celery, Redis.
- **DevOps**: Docker, Docker Compose.
- **Desktop**: Python, PyQt5.
